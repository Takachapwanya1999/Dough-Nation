# Dough Nations Clocking System

## Features
- Track work hours, breaks, and overtime.
- Generate payroll reports.
- User Management System for profile CRUD operations.
- Login authentication.

## Tech Stack
- **Front-end:** HTML, CSS, JavaScript (React or Vue.js)
- **Back-end:** Node.js with Express.js
- **Database:** MongoDB with Mongoose

## Compliance
- Ensure compliance with South African labor laws (BCEA).
